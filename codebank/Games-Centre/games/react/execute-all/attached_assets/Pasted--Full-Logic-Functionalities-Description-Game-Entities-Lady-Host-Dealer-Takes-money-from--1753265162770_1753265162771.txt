🧠 Full Logic & Functionalities Description
🧍 Game Entities:
Lady Host (Dealer):

Takes money from each player (coded form).

Exchanges it into gold/silver bars (based on amount paid).

Distributes cards equally to all players.

Announces the start, declares winner, restarts game.

Players (2–6 multiplayer):

Can join table with a unique code or invite link.

Deposit money (coded-money) to get gold/silver bars.

Take turns playing a card each round.

Winner gains all bars on the table.

Game Room (Table):

Handles room ID, turn control, player synchronization.

Chat or emote support for social interaction.

Coded-Money System:

Accepts player deposits (e.g., in-game credits).

Exchanges them into gold bars (large bet) and silver bars (small bet).

Card Game Rules (example variant):

1 deck (52 cards)

Highest card wins (you can add complex rules later)

Players reveal one card per round.

Continue until cards finish or player wins the pot.
